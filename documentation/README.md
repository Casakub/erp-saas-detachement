# Documentation (Non Contractuelle)

## Objectif

- Centraliser la documentation non contractuelle du projet.
- Garder les documents LOCKED existants à leur emplacement actuel.

## Règles

- Ne pas déplacer les fichiers du dossier `ERP Détachement europe/`.
- Utiliser ce dossier pour les guides opérationnels, notes internes et supports d'exécution.
- Conserver l'approche contract-first: contrat d'abord, implémentation ensuite.

## Structure

- `documentation/runbooks/` : procédures opérationnelles et guides d'utilisation.
