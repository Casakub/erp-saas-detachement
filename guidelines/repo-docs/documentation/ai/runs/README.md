# Run Packs IA

## Objectif

- Fournir des packs préremplis par lot/module.
- Réduire le coût de contexte avant implémentation.
- Garder l'exécution alignée aux contrats LOCKED.

## Utilisation

1. Ouvrir le pack du lot/module visé.
2. Vérifier le scope endpoint/table/event/rbac.
3. Compléter les champs run spécifiques (ticket, owner, branche).
4. Exécuter le run IA avec ce pack comme contexte principal.
