# Runbooks

## Objectif

- Documenter les procédures d'exploitation non contractuelles.
- Fournir des étapes reproductibles pour l'équipe.

## Exemples de runbooks à ajouter

- Connexion et usage MCP (Figma, Supabase).
- Procédures locales de vérification.
- Dépannage des environnements de travail.
