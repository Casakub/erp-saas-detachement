# Design Tokens - Typography

## Rule

- Definir la typographie en tokens semantiques.

## Token Families

- `font.family.primary`
- `font.size.xs|sm|md|lg|xl|2xl`
- `font.weight.regular|medium|semibold|bold`
- `font.lineheight.tight|normal|relaxed`

## Usage

- Titres: roles semantiques, pas valeurs ad hoc.
- Corps: lisibilite prioritaire pour interfaces data-heavy.
- Mobile: garder la lisibilite sans surcharge.
