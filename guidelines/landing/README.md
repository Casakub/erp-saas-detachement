# Landing Pack - Entry Point

## Objectif

- Finaliser la documentation de la landing avant execution IA.
- Partir de l etat visuel actuel (captures) sans casser les contraintes contract-first.
- Standardiser le run Figma Make pour la landing.

## Fichiers du pack

1. `guidelines/landing/landing-current-state-audit.md`
2. `guidelines/landing/landing-run-prompt.md`
3. `guidelines/landing/landing-handoff-checklist.md`
4. `guidelines/landing/landing-refactor-plan.md`

## Ordre d usage

1. Lire l audit actuel.
2. Lancer la session avec le prompt de run.
3. Executer section par section selon le refactor plan.
4. Valider avec la handoff checklist.
