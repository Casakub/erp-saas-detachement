# Landing - Refactor Plan (Section By Section)

## Scope

- Scope unique: landing marketing.
- Module de reference pour mapping: `M13` (surfaces CMS/marketing/orchestration UI).
- Hors scope: logique metier backend decisionnelle.

## Plan d execution

1. Header + Navigation
- harmoniser nav, CTA header, etat sticky.
- livrer variants: desktop/mobile + menu ouvert/ferme.

2. Hero
- verrouiller hierarchie titre, proof points, stats, CTA duo.
- normaliser carte visuelle produit.

3. Features Grid
- uniformiser dimensions cards, marges, tags, alignements.
- verifier lisibilite de tous textes secondaires.

4. Integrations
- standardiser composants logo-card.
- definir etats hover/focus/selected si necessaire.

5. Testimonials
- normaliser card temoignage + controles carousel.
- definir etats active/inactive/focus des bullets et fleches.

6. CTA Final
- aligner style et promesse avec hero.
- verifier lisibilite et priorite primaire/secondaire.

7. Footer
- harmoniser colonnes, titres, liens, baseline legale.
- verifier lisibilite du texte bas de page.
