# Components - Base Library Target

## Base Components To Maintain

- Buttons
- Inputs
- Selects
- Badges
- Alerts
- Cards
- Tables
- Modals
- Tabs
- Pagination
- Empty states
- Loading states

## Responsive Requirement

- Chaque composant critique doit avoir une variante desktop et mobile.

## Accessibility Minimum

- Contraste lisible.
- Etats focus visibles.
- Tailles de cibles tactiles coherentes sur mobile.
